hudamin 2014-08-22

文档的作用

  1. 写文档于己是一种总结和梳理
  2. 写文档与他是一种知识分享和传承

什么会影响我们写文档的体验

  1. 排版：缩进、段落、字号、字体、颜色、源码、图片、链接等
  2. 显示面积：强制换行、均匀分布，太不像代码了

  1. 接口文档：字处理软件输出的接口文档很不友好
    • 小尾巴，拼写纠正，输入纠正，非等宽字体。—— 太智能了，不见得好啊

    • 图文混排，没法直接拷贝

  1. 更新：如何保证让他人总能看到最新版的文档？

    甲：那个接口貌似不能用，帮忙看下

    乙：好的，我看下，稍等

  1. 分享：U盘？邮件？

HTML格式Web文档

怎么写HTML文档？

  1. 手写？FE都看着头疼

  2. 编辑工具转存？前提是你还得自己排版

我们希望的方式

  1. 不直接写HTML标签
  2. 纯文本，格式简单
  3. 输出为HTML文档
  4. 支持主题随意切换

markdown

  1. JOHN GRUBER发明，帮助简单快速编写web文档
  2. 特定格式书写纯文本文档：使用特殊字符和缩进，通过工具就可以转换成美观的HTML文档
  3. 解析器有很多，除了JOHN GRUBER提供的Perl解析器外，互联网上还有很多其他版本的，

    比如Python版本、Nodejs版本、java版本、C版本等等

  4. 新闻文档平台使用的markdown解析器是我前段时间开发的一个基于lex-yacc的编译型C解析器

如何使用markdown写web文档

例子1：标题＋段落

一级标题

前言：段落一文本

二级标题

段落二文本

三级标题

段落三文本

写法：

# 一级标题
前言：段落一文本
## 二级标题
段落二文本
### 三级标题
段落三文本

例子2：标题＋段落＋代码

以下是一段C代码

简介C代码功能

int i = 0;
for(;i<10;i++){
    printf("%d",i);
}

总结段落

写法：

### 以下是一段C代码
简介C代码功能
    int i = 0;
    for(;i<10;i++){
        printf("%d",i);
    }

编写注意事项

  1. 纯文本编辑器，ultraEdit, notepad++, editplus, webstorm, vim等，不要用word
  2. 保存编码：UTF-8
  3. 文件扩展名：.md

如何使用百度新闻文档平台

  1. 浏览器访问：http://db-news-fe2.vm.baidu.com:8520/docs/
  2. SVN，建立自己的子目录：http://db-news-fe2.vm.baidu.com:8520/svn/newsdocs
  3. 本地预览
    > markdown xxx.md
  4. 编辑index.md，添加你的文档到首页
