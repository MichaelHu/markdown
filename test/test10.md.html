HTML格式Web文档

@s

怎么写HTML文档？

  1. 手写？FE都看着头疼

  2. 编辑工具转存？前提是你还得自己排版

@s

我们希望的方式

  1. 不直接写HTML标签
  2. 纯文本，格式简单
  3. 输出为HTML文档
  4. 支持主题随意切换

@s

markdown

  1. JOHN GRUBER发明，帮助简单快速编写web文档
  2. 特定格式书写纯文本文档：使用特殊字符和缩进，通过工具就可以转换成美观的HTML文档
  3. 解析器有很多，除了JOHN GRUBER提供的Perl解析器外，互联网上还有很多其他版本的，

    比如Python版本、Nodejs版本、java版本、C版本等等

  4. 新闻文档平台使用的markdown解析器是我前段时间开发的一个基于lex-yacc的编译型C解析器

编写注意事项

  1. 纯文本编辑器，ultraEdit, notepad++, editplus, webstorm, vim等，不要用word
  2. 保存编码：UTF-8
  3. 文件扩展名：.md

如何使用百度新闻文档平台

  1. 浏览器访问：http://db-news-fe2.vm.baidu.com:8520/docs/
  2. SVN，建立自己的子目录：http://db-news-fe2.vm.baidu.com:8520/svn/newsdocs
  3. 本地预览
    > markdown xxx.md
  4. 编辑index.md，添加你的文档到首页
